Instalación del Servidor TFTP

#Descargar Servidor TFTP

Para este caso se utilizará el programa Tftp32. Ir a la página oficial y descargar la versión más reciente.

Al hacer clic en el botón direct link se descarga un archivo Tftp32-4.64-setup.exe (541 Kb aprox.). Una vez descargado el archivo, se ejecuta para empezar la instalación.

#Instalación

Al ejecutar el archivo se mostrará la licencia del programa. Clic en I Agree para aceptar los términos y condiciones del programa.

En la siguiente pantalla se mostraran los componentes que se van a instalar en el equipo. Clic en Next para continuar.

Elegir el directorio donde se instalará el programa y hacer clic en Install para continuar.

Poco después empezará a descomprimir los archivos necesarios para la instalación del programa.

Una vez termine la instalación, se podrá cerrar la ventana. Clic en Close para finalizar la instalación.
